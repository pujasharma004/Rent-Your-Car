# Rent-Your-Car
A modern and responsive car rental landing page built using HTML, CSS, and JavaScript, featuring interactive UI components and a premium user experience

## ✨ Features

- 🚘 Car showcase section  
- 💎 Premium UI design (Glass Morphism)  
- 📱 Fully responsive layout  
- 💰 Pricing section for different car categories  
- 📅 Booking section with user input fields  
- ⭐ Testimonials section  
- ❓ FAQ accordion with smooth animations  
- 🎯 Interactive UI using JavaScript

## 🛠️ Tech Stack
